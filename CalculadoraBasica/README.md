##**Calculadora Básica das 4 operações Matemáticas**
Uma calculadora simples das **4 operações básicas** da matemática, com interação com o usuário via terminal. 
💡 Solução básica para as primeiras implementações usando a linguagem de programação Java ☕.
✅ Foi aplicado _switch case_ para execução de ações com base nos dados enviados pelo usuário, que neste caso foi feito pelo operadores (**+  -  *  /**), e não pelos números.
